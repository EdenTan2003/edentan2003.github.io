Membership Loyalty Program using Terminal with Javascript

In this project, I have created a membership record using Javascript, which allows the user (admin) to first key in their password followed by their name and they will be able to view all member's info including adding and deleting of member's in the record. Furthermore, there is also statistics to allow the admin to know the youngest and oldest member, total members in each membership type etc. This project is done using Javascript, HTML and CSS.

GoRecipe Mobile Application using react-native

In this project, I have created a mobile application that allows users to view recipes that they are interested in like the ingredients, steps, time taken, servings etc. They would be able to login using either a username/password or they can use their google account to login. Furthermore, they would be able to add recipes to their favorites and they can view it in their favorites page, including the creation and deletion of their own recipes.
